### CSK Shopping Application


#### Application Features
* Admin
  * Add product
  * Update product
  * Delete product
  * View/Approve/Decline Order
* Customer
  * Register user
  * User login
  * Update Address
  * Add to Cart
  * Update/ Delete Product
  * Place order

* Technologies: 
  * Angular 11
  * Typescript
  
### Build & Deployement

To build project, Run `ng build`

To deploy project, Run `ng serve` 

Application url `http://localhost:4200/`
